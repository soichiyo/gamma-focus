# Gamma Focus

40Hz バイノーラルビートジェネレーター — 集中力を高めるためのアンビエントサウンドミキサー。
<img width="884" height="1384" alt="CleanShot 2026-04-15 at 12 33 50@2x" src="https://github.com/user-attachments/assets/1333a8f8-62d3-41d0-89d6-cd8012684151" />

https://gamma-focus.vercel.app/

ヘッドホンを装着し、再生ボタンを押すだけで 200Hz / 240Hz のバイノーラルビートによる 40Hz ガンマ波を生成します。雨音・波・カフェノイズなどのアンビエントレイヤーを重ねて、自分好みの集中環境を作れます。

## 技術スタック

- Next.js 16 / React 19
- Tailwind CSS 4
- Web Audio API（バイノーラルビート生成）
- Lucide React（アイコン）
