# Gamma Focus

40Hz バイノーラルビートジェネレーター — 集中力を高めるためのアンビエントサウンドミキサー。

https://gamma-focus.vercel.app/

ヘッドホンを装着し、再生ボタンを押すだけで 200Hz / 240Hz のバイノーラルビートによる 40Hz ガンマ波を生成します。雨音・波・カフェノイズなどのアンビエントレイヤーを重ねて、自分好みの集中環境を作れます。

## 集中セッション OS

サウンドミキサーの上に、ミッション単位の集中セッション層を備えています。

- 一行ミッションを入力し、目標時間（15 / 25 / 45 分、またはフロー＝無制限）を選んで開始
- 経過時間と目標までの進捗を表示。一時停止・再開・完了（レビュー）・中断（同じくレビューに保存）に対応
- セッション中もバイノーラル／アンビエントの音量操作はそのまま利用可能
- 完了・中断後に集中度（1〜5）・結果・メモを残す軽量レビュー
- 直近のセッション履歴をローカルに保存（最大 20 件）

## アダプティブ・オーディオコーチ v1

監視や外部センサーに頼らず、本人のチェックインと時間ベースの軽いナッジで集中を支えます。

- 一定間隔でチェックインを提示し、focused / drifting / break を記録
- drifting 時はホワイトノイズを穏やかに引き上げ（ミキサー上で可視・可逆）
- break 時はブレイクタイマーへ切り替え（詳細は次節）
- チェックインと介入はセッションログに記録

## Break Awareness v1

「休憩したい」のチェックインから、見落としにくいブレイクタイマーへ切り替えます。

- 3 / 5 / 10 分のブレイク時間を選び、選択した時点でオーディオを停止しセッションを一時停止
- ブレイク中はタブタイトルに残り時間、対応環境ではアプリバッジも更新（任意でブラウザ通知の許可も求める）
- タイマー終了時に通知・サウンドキュー・`Break ended` 表示で知らせる
- `Return to Focus` でブレイクをクリアし、セッションを再開

すべてローカル（`localStorage`）で完結します。デスクトップ監視・ブラウザ監視・集中スコアの自動算出はこのリポジトリでは行いません（将来 `log-my-pc` などの外部シグナルと連携する余地のみ残しています）。

## 技術スタック

- Next.js 16 / React 19
- Tailwind CSS 4
- Web Audio API（バイノーラルビート生成）
- Lucide React（アイコン）

- Vitest（純粋ロジックとストレージのテスト）
